 *************Commandes************


À travers ces différentes commandes, tu dois pouvoir déplacer ton robot, exécuter les instructions données avec le fichier présent dans une pièce. 


LINK “arg” :   Se déplacer à la pièce suivante 
GRAB “nom du fichier” : Récupérer le fichier et lire sa valeur  
JUMP “n” : Permet de sauter n instruction (n entier)  
HALT : Rupture de séquence
Entrée : Pour taper l'instruction suivante et passer à l'instruction suivante
r :  Recommencer
s : Accélérer 
Q | q : Abandonner/Quitter la mission


Tu peux effectuer des opérations élémentaires de l’arg1, arg2 et stock la valeur dans le registre 
ADDI “arg1” “arg2” “registre” : pour additionner  
MULI “arg1” “arg2” “registre” : pour multiplier 
SUBI “arg1” “arg2” “registre” : pour soustraire 
DIVI “arg1” “arg2” “registre” : pour diviser 
MODI “arg1" “arg2” “registre” : pour effectuer le reste de la division entière de arg1 par arg2

 ***********Gameplay***********


Ton rôle est d’accomplir les différentes missions qui te sont données grâce à un robot Ninja. Tu dois pour cela te déplacer dans la pièce où se trouve le fichier secret et le décoder en effectuant les instructions qui te sont donné.
